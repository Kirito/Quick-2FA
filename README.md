# Quick 2FA
Lightweight Chrome / Brave extension for generating TOTP codes from your browser, That I made to learn extension making.

designed to be offline, with no accounts and no tracking
uses **chrome.storage.local** to store data (data doesnt leave your browser)


## Installation
1. clone or download repo
2. open chrome or brave and head to either `brave://extensions/` or `chrome://extensions`
3. enable developer mode
4. click on load unpacked
5. select the Quick 2FA folder and done.

## Features 
- import and export saved 2FA secrets and profile names
- offers a blur feature that works on the 2FA secret input and the 2FA code
- fully offline and only stores 2FA profiles on your browser
